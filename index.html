Política de Privacidad

Última actualización: Abril 2025

1. Información que recopilamos

NUVIs recopila información necesaria para proveer el servicio de asistencia conversacional automatizada, incluyendo:

  • Número de teléfono y nombre del contacto
  • Contenido de los mensajes enviados al bot
  • ID de usuario de Instagram (cuando aplica)
  • Historial de conversaciones para mantener el contexto

2. Cómo usamos la información

Usamos la información exclusivamente para proveer respuestas automatizadas personalizadas, mejorar la calidad del servicio y generar reportes para el cliente empresarial.

3. Proveedores de infraestructura

  • Anthropic — Procesamiento de lenguaje natural (Claude AI)
  • Meta Platforms — WhatsApp Business API e Instagram API
  • Upstash — Almacenamiento temporal de historial (Redis)
  • Supabase — Base de datos del panel
  • Vercel — Infraestructura de servidor

4. Uso de la API de Meta

NUVIs utiliza la API de WhatsApp Business y la API de Instagram de Meta. Los datos no se usan para publicidad ni se comparten con terceros no autorizados.

5. Retención de datos

El historial de conversaciones se almacena 3 días. Los datos del panel se almacenan mientras el cliente mantenga el servicio activo.

6. Derechos del usuario

Podés solicitar acceso, rectificación o eliminación de tus datos contactando a: privacynuvem@gmail.com

Términos de Servicio

Última actualización: Abril 2025

1. Aceptación

Al usar NUVIs, aceptás estos términos.

2. Descripción del servicio

NUVIs provee un servicio de asistencia conversacional automatizada mediante IA, integrado con WhatsApp Business API e Instagram API de Meta.

3. Uso aceptable

Queda prohibido usar NUVIs para enviar spam, contenido ilegal, o cualquier uso que viole las políticas de Meta.

4. Responsabilidades del cliente

  • Cumplir con las políticas de WhatsApp Business y Meta
  • Obtener el consentimiento de sus usuarios para recibir mensajes automatizados
  • El contenido configurado en el bot es responsabilidad del cliente
